
servlet 3.x upload:

1:upload by servlet:
http://localhost:8080/uploader/forward2UploadServlet

2:upload by spring mvc(single or multi file):
http://localhost:8080/uploader/forward2UploadSpringMvc

3:upload by ajax/html5/spring mvc(single or multi file)
http://localhost:8080/uploader/forward2UploadAjax

4:upload by ajax/html5/spring mvc/progress single file
http://localhost:8080/uploader/forward2UploadProgressSingle

5:upload by ajax/html5/spring mvc/progress multi file
http://localhost:8080/uploader/forward2UploadProgressMulti
